# 🧁 Study Bakery

Study Bakery is a cute aesthetic Pomodoro-style study app built using Electron, HTML, CSS, and JavaScript ✨

It helps users stay productive while making studying fun by turning completed study sessions into adorable bakery rewards 🍰

---

## 💖 Features

- ⏳ Pomodoro study timer
- ☕ Automatic break timer
- 🍰 Bakery shelf reward system
- 🔥 Daily streak tracker
- 🎯 Daily study goals
- 📊 Weekly progress chart
- 🎶 Background music support
- 💌 Motivation quotes
- 🌸 Cute pastel bakery UI
- 💾 Local storage support

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Electron.js

---

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/study-bakery.git
```

### 2. Open project folder

```bash
cd study-bakery
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the app

```bash
npm start
```

---

## 📦 Build App

To create the desktop app:

```bash
npm run build
```
